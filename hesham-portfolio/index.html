Flagship Project

Face Generation from Textual Description

A graduation project that generates realistic human faces from a written description — turning a sentence like "a young woman with curly hair and glasses" into a photorealistic portrait.

Built on StyleGAN2 for image synthesis and CLIP to align generated faces with the input text, with an LLM layer to interpret and expand loose descriptions. Output quality was pushed further with RAG-based super-resolution, wrapped in an interactive Streamlit app deployed on Google Colab.
